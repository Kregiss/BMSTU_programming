## Лаба 1

Выполнить распечатку GIMPLE/IR из gcc компилятора. Например, это можно сделать из `./gcc-4.9.0/gcc/tree-ssa-dce.c`. Как вариант, можно сделать распечатку из плагина для GCC. На ваше усмотрение.
 

Эта распечатка должна в себя включать следующее:

* базовые блоки, блоки-предшественники, блоки-последователи

* содержимое базовых блоков (GIMPLE инструкции):
    - арифметические операции
    - Ф-функции
    - инструкции ветвления
    - выражения доступа в память (`ArrayRef` и др, которые были в лекции)

для распечатки использовать только `printf` и нельзя использовать встроенные средства распечатки всего IR или его фрагментов.

Т.е. смысл лабы - пощупать IR, поиграться с ним
